Definition and Examples

  • Definition: An interval is a set of real numbers

  • Notation: We use brackets and parentheses

  • Examples:

  • [a, b]

  • (a, b)

  • (a, ∞)

More Examples

  • [0, 1] = {x : 0 ≤ x ≤ 1}

  • (−1, 2) = {x : −1 < x < 2}

  • (3, ∞) = {x : x > 3}

  • (−∞, 0] = {x : x ≤ 0}

  • (0, 1] = {x : 0 < x ≤ 1}

  • [−2, 2) = {x : −2 ≤ x < 2}

Properties

  • Intersection: [a, b] ∩ [c, d] = [max (a, c), min (b, d)]

  • Union: Not always an interval

  • Length: length([a, b]) = b − a

  • Center: $\text{center}([a,b]) = \frac{a+b}{2}$

  • Subset: If a ≤ c ≤ d ≤ b, then [c, d] ⊆ [a, b]

Graphical Representation On the real line:

  • Closed endpoint: •

  • Open endpoint: ∘

  • Infinite: → or ←

Examples:

  • [1, 3]: •− − −•

  • (1, 3): ∘− − −∘

  • (2, ∞): ∘→

  • (−∞, 1]: ←•

Applications - Solving Inequalities

Example 1 Solve: 2x − 3 < 5

  • 2x − 3 < 5

  • 2x < 8

  • x < 4

  • Solution: (−∞, 4)

Applications - Set Operations

Example 2 Find: [−1, 3] ∩ (2, 5)

  • [−1, 3] = {x : −1 ≤ x ≤ 3}

  • (2, 5) = {x : 2 < x < 5}

  • Intersection: {x : 2 < x ≤ 3}

  • Result: (2, 3]

Applications - Function Domains

Example 3 Find domain of: $f(x) = \sqrt{4 - x^2}$

  • Need: 4 − x2 ≥ 0

  • x2 ≤ 4

  • −2 ≤ x ≤ 2

  • Domain: [−2, 2]

Important Theorems

  • Intermediate Value Theorem: If f continuous on [a, b], takes all values between f(a) and f(b)

  • Extreme Value Theorem: Continuous on [a, b] ⇒ attains max and min

  • Mean Value Theorem: If f differentiable on (a, b), ∃c ∈ (a, b) with: $$f'(c) = \frac{f(b)-f(a)}{b-a}$$

Summary

  • Intervals: sets of real numbers

  • Notation: [ for included, ( for excluded

  • Types: closed, open, half-open, infinite

  • Properties: intersection, length, center

  • Applications: inequalities, domains, theorems

  • Graphical: •, ∘, →

  • Fundamental in real analysis

  • Essential for calculus

  • Basis for advanced mathematics
